## ?? Development Workflow

### Making Frontend Changes

```
1. Edit file in client-app/src/
        ?
2. Vite detects change
        ?
3. HMR updates browser
        ?
4. See changes instantly! ?
   (No page refresh needed)
```

### Making Backend Changes

```
1. Edit file in Core/, Hubs/, or Managers/
        ?
2. If using "dotnet watch run":
   Server auto-restarts
        ?
3. If using "dotnet run":
   Stop (Ctrl+C) and restart manually
        ?
4. Changes applied!
```

## ?? Data Flow Diagram

### User Registration Flow

```
User enters ID
      ?
Vue Component (VideoChat.vue)
      ?
SignalR Service (signalr.service.ts)
      ?
WebSocket Connection
      ?
SignalR Hub (VideoCallHub.cs)
      ?
User Manager (UserManager.cs)
      ?
Store user in memory
      ?
Broadcast user list to all clients
      ?
All connected clients receive update
      ?
Vue components update UI
```

### Video Call Flow

```
User clicks "Call" button
      ?
WebRTC Service creates offer
      ?
Send offer via SignalR
      ?
SignalR Hub forwards to target user
      ?
Target user receives offer
      ?
Target user creates answer
      ?
Send answer via SignalR
      ?
Original user receives answer
      ?
ICE candidates exchanged
      ?
Peer-to-peer connection established
      ?
Video/audio streams directly between peers
```
